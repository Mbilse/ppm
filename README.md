Meteor 是一个演示文稿制作器（即大家说的 "PPT"）。

作为亮点，Meteor 生成的演示文稿可**随时随地**播放。即，在其它电脑上，不必安装 Meteor，演示文稿也能正常播放。

目前，功能上能实现基本的制作，可满足日常使用。

（界面有变更，以实际为准）
![编辑界面](art/edit.png)
![播放界面](art/play.png)

## 使用方法

- 在右侧 Releases 中下载最新版本的可执行文件（`.exe` 文件）。
- 运行程序，在浏览器中打开显示的网址，即可使用。

## 运行方法

安装 python，安装依赖，运行命令

```cmd
python manage.py runserver
```
即可。
